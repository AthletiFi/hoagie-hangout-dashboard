Strategic Implications

The data shows a clear bifurcation in user needs based on age, suggesting three potential strategic paths:

  1. Focus on 16-year-olds: Build recruitment-focused features first, emphasizing professional presentation and coach interactions
  2. Focus on 13-year-olds: Build engagement and social features first, emphasizing gamification and friend interactions
  3. Dual-track approach: Develop separate feature sets for different age groups with a common core platform
Recommended Path (based on focus‑group data)

Start with the segment that has an immediate “job to be done”:

  • Phase 1 (Q2): 16‑yos → ship recruiting‑centric features (highlights, coach messaging, pro card designs).
  • Phase 2 (Q3): Layer in 13‑yo gamification (feed, fantasy roster, follow/friend system).
  • Shared Core: smoother sign‑up, privacy controls, improved card editor.

Strategic Decisions

A. Age Segmentation

✅ Prioritize 16‑yos first, but design a gamified path for 13‑yos (see Age‑Insights).

B. Card Design Evolution

Ship 3 “clean / pro” templates ASAP; keep current fun templates for younger users.

C. Team Video Integration

Fast‑track full‑match uploads & auto‑clip selection; it underpins both recruiting & social feed.

D. Onboarding Simplification

Add phone‑based sign‑up + one‑tap magic‑link; move heavy email verification to coaches.

E. Privacy / Audience Control

Build profile visibility toggles (public / friends / coaches) before public launch.

F. Branding / Domain

Begin name‑change discovery this month; new short URL is part of onboarding fix.
